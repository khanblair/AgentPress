# Document Ledger

A log of all documents generated by the AgentPress pipeline. Each entry is created when a pipeline run completes successfully.

## Entry Format

Each ledger entry records:

| Field | Description |
|---|---|
| Session ID | Unique identifier for the pipeline run |
| Document Title | Human-readable filename slug |
| Output Format | `pptx` · `docx` · `xlsx` · `pdf` |
| QA Retries | Number of Designer→Inspector loops |
| Evolution | Whether the Meta-Engineer was triggered |
| File Size | Final output file size |
| Timestamp | UTC completion time |

## QA Pass Rate

The Inspector runs a two-stage review on every document:

- **Stage 1** — Factual accuracy against research data
- **Stage 2** — Brand compliance against `brand.md`

A document is only delivered when both stages pass and the file was built successfully.

---

*Entries are appended automatically after each completed pipeline run.*
